# Elvra OSINT   
### Identity Intelligence & OSINT Analyzer  

A lightweight **Python-based OSINT intelligence tool** for analyzing usernames, emails, phone numbers, and correlating digital identity data across multiple sources.

## ☁️ Overview  

**Elvra OSINT** is a simple identity intelligence tool :

It provides:
- Username / email / phone enrichment
- Multi-source OSINT search
- Identity correlation engine
- Identity confidence scoring
- Graph-based relationship mapping
- Web dashboard interface

## 📄 Features

* Multi-source OSINT aggregation
* Username / email / phone analysis
* Identity fusion system
* Confidence scoring (0–100)
* AI-based analysis layer
* Graph visualization system
* Radar security overview
* Web interface (FastAPI dashboard)

## 📁 Output Files

* report.json → OSINT structured results
* identity_fusion → merged identity data
* graph nodes → relationship mapping
* confidence score → identity reliability metric
